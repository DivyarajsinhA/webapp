# 🎓 B.Tech Student Management System

A full-stack web application built for managing university student records. This project was developed as part of a B.Tech Information Technology curriculum to demonstrate fundamental CRUD operations, database management, and responsive UI design.

## 🚀 Features
- **Create:** Register new students with their Enrollment ID, Branch, and CGPA.
- **Read:** View a dynamic, real-time table of all enrolled students.
- **Delete:** Remove graduated or transferred students from the database.
- **Responsive Design:** Styled with Tailwind CSS for mobile and desktop viewing.

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **Database:** SQLite3
- **Frontend:** HTML5, Tailwind CSS, Jinja2 Templating
